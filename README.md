# [glitchdeploy.com](http://glitchdeploy.com/)

## Development

Clone repository.

```sh
$ git clone https://github.com/distkloc/distkloc.github.io.git
$ cd distkloc.github.io
```

Install node packages.

```sh
$ npm install
```

Run gulp task.

```sh
$ gulp
```

## Deployment

```sh
$ gulp deploy
```

This command pushes dist directory to origin master.
